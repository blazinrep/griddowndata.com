## Official Logo Rule

The **mountain-and-tower logo** is the official, unified GridDownData mark. Use it consistently across:

- All storefront imagery and marketing renders (`index.html` assets)
- The offline USB dashboard (`START_HERE.html` assets)
- Printed quick-reference cards, Faraday pouch branding, and product packaging

When in doubt, lead with the mountain-and-tower logo. Do not mix unrelated iconography or create alternate marks for the same brand touchpoint.

All paths are referenced from the repository root, e.g. `assets/images/hero-faraday-pouch-closeup.jpg`.
